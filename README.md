Repositório de uma coleção dos meus primeiros projetos práticos de desenvolvimento front-end, contento desde sites estáticos com interface simples até integrações com APIs públicas.
Cada projeto está separado em uma pasta e pode ser executado de forma independente.

## 🚀 Tecnologias

- HTML5
- CSS3
- JavaScript

## 📁 Estrutura do Repositório

Cada pasta desse repositório contém um projeto específico.

| Projeto | Descrição |
|---|---|
| [Audiobook](./Audiobook) | Player de audiobook com controles de reprodução de áudio |
| [Buscar CEP](./Buscar%20CEP) | Consulta de endereço a partir de um CEP via API ViaCEP |
| [Consultar Clima](./Consultar%20Clima) | Exibição de condições climáticas de uma cidade via API OpenWeather |
| [Gerador de Senhas](./Gerador%20de%20Senhas) | Geração de senhas seguras com opções de customização |
| [Lista de Tarefas](./Lista%20de%20Tarefas) | Gerenciamento de tarefas com adição, conclusão e remoção de itens |
| [Login e Cadastro](./Login%20e%20Cadastro) | Formulários de autenticação e login com validação de campos |
| [Troca de Temas](./Troca%20de%20Temas) | Alternância entre temas claro e escuro (dark/light mode) |

### Dentro dos Projetos

```
Nome do Projeto/
├── assets/       # Arquivos de ícones ou imagens
├── script/       # Arquivo de script da página (.js)
├── style/        # Arquivo de estilização (.css)
└── index.html    # Arquivo da página inicial do site
```

## 🛠️ Como executar

Cada projeto é composto por arquivos estáticos (`HTML`, `CSS` e `JS`), sem necessidade de instalação ou build.

1. Clone o repositório:
```bash
git clone https://github.com/rafaelbaratta/sites-front-end.git
```

2. Acesse a pasta do projeto desejado:
```bash
cd sites-front-end/<nome-do-projeto>
```

3. Abra o arquivo `index.html` diretamente no navegador.
